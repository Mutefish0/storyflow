# storyflow - 极客的笔记本
极客的笔记本，使书写Markdown变得更愉悦

## 技术栈
- Electron
- React
- Typescript
- Glup & Rollup

## 运行平台
暂只支持Mac

## 特性（暂只实现部分
- Markdown 语法高亮
- 代码高亮
- 可双向编辑
- 支持导出为PDF或HTML文件

## 其他
本软件离不开以下优秀开源库的支持：
- [PEG.js: Parser generator for JavaScript](http://pegjs.org/)
- [highlight.js - Syntax highlighting for the Web](https://highlightjs.org/)
